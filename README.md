### 🔬 Comparative Sentiment Analysis — Sparse vs Dense Embeddings
> `NLP` · `Embedding Benchmarking` · `Deep Error Analysis`

**What it actually does:** End-to-end NLP pipeline on **50,000 IMDB reviews** that rigorously benchmarks two paradigms — not just runs them.

| Approach | Method | Result |
|---|---|---|
| Sparse | TF-IDF + Logistic Regression | Baseline |
| Dense | Word2Vec / FastText + LSTM | **89.3% accuracy** |

**What made it non-trivial:**
- 📐 t-SNE visualizations to validate embedding cluster separability — not just accuracy scores
- 🔍 Deep error analysis on **negation**, **contrastive conjunctions** (`but`, `however`), and **mixed-sentiment** reviews
- 🧩 Investigated *where* each method fails systematically, not just overall metrics

`Python` · `Scikit-learn` · `Gensim` · `TensorFlow/Keras` · `Matplotlib`
