## Projects

### 🔬 Comparative Sentiment Analysis — Sparse vs Dense Embeddings
> *NLP · Embedding Methods · Deep Error Analysis*

Built a full NLP pipeline on 50,000 IMDB reviews to rigorously compare traditional and neural text representations.

| Approach | Method | Accuracy |
|----------|--------|----------|
| Sparse | TF-IDF + Logistic Regression | Baseline |
| Dense | Word2Vec/FastText + LSTM | **89.3%** |

**What made this non-trivial:**
- Used t-SNE to visualize embedding cluster separability and validate representational quality
- Deep error analysis across linguistic edge cases: negation, contrastive conjunctions, mixed-sentiment reviews
- Investigated *where* each approach fails — not just overall metrics

`Python` `Scikit-learn` `Gensim` `TensorFlow/Keras` `Matplotlib`

---

### 🧠 Autism Spectrum Disorder Prediction
> *Classification · Imbalanced Learning · Clinical Data*

End-to-end classification pipeline on behavioral and demographic data, designed with reliability as the primary constraint.

**Key decisions:**
- Applied SMOTE to address class imbalance — prioritized recall to minimize false negatives in a sensitive context
- Tuned Random Forest and XGBoost with cross-validated hyperparameter search
- Evaluated beyond accuracy: precision-recall tradeoffs, confusion matrix analysis

`Python` `Scikit-learn` `XGBoost` `imbalanced-learn`

### 📊 Medical Insurance Cost Prediction
> *Regression · Feature Engineering · EDA*

Regression pipeline predicting insurance costs from demographic and health indicators.

**Focus areas:**
- In-depth EDA to surface feature interactions and distribution skews
- Categorical encoding strategy selection (ordinal vs one-hot based on cardinality)
- Model evaluation using R² and residual analysis
